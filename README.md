# Project H

## Dependencies

-   Nuxt 3
-   Vue 3

# Install the dependencies

-   `npm install`

## Commands

-   `npm run dev` - Starts the development server
-   `npm run build` - Builds the project for production
-   `npm run start` - Starts the production server
